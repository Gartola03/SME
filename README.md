# SME

Este repositorio contiene una serie de funciones implementadas tanto en R como en Python, desarrolladas para la asignatura de Software Matemático y Estadístico (SME) del máster en Koputazio Ingeniaritza eta Sistema Adimentsua (KISA).

Su objetivo principal es servir como introducción al mundo de la programación, especialmente para quienes tienen experiencia limitada o muy específica en este ámbito. Además, el curso abarca desde conceptos básicos hasta temas avanzados en ambos lenguajes de programación, que son los más habituales y se utilizarán de manera intensiva a lo largo del máster.
